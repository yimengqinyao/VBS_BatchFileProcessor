最早手里只有一个能获取单个文件MD5的VBScript脚本，每次手动输入路径查文件哈希值都要折腾半天，实在嫌麻烦，就靠着半吊子VBScript知识自己捣鼓出第一版批量获取的代码。

### 失控：AI接管的摸鱼时光
写出第一版批量代码后，顺手丢给AI让它帮忙分析优化逻辑、修复语法问题。结果改着改着突然闲得发慌，干脆让AI顺着它自己的建议一路迭代加新功能：
- 今天加个读取文件MD5的功能
- 明天整个导出结果到CSV的模块
- 后天又加了个批量重命名的无关功能
  
一来二去，就折腾出了这么个功能杂乱的脚本仓库。

### 现状：AI产物
